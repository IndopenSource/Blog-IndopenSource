Blog IndopenSource disiapkan sebagai tempat mencatat kerja komunitas open source
Indonesia secara rapi dan mudah ditinjau.

## Fokus Awal

Fokus awal blog ini adalah tulisan yang membantu pembaca menemukan proyek,
memahami praktik perawatan open source, dan mengikuti perkembangan komunitas.

Kategori awal yang akan dikurasi:

- Project spotlight.
- Learning notes.
- Release notes.
- Catatan maintenance.
- Kabar komunitas.

## Cara Berkontribusi

Kontributor dapat menyalin template dari `templates/article.md`, membuat file di
folder `content/YYYY/MM/`, lalu membuka pull request.

Setiap tulisan akan direview agar struktur, sumber, dan nada tulisannya sesuai
dengan kebutuhan pembaca IndopenSource.

## Langkah Berikutnya

Setelah alur editorial stabil, konten dari repo ini dapat diintegrasikan ke
website `indopensource.org`.
